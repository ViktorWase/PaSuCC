    P.A.SU.C.C

    P.A.SU.C.C. is a Polyalphabetical Substitution Cipher Cracker. It runs on Matlab and uses an evolutionary algorithm that optimizes fitness of the keys. The fitness is a measure of how well the frequency of the letters and letter pairs of the plaintext matches that of english.

    The code includes the three Beale ciphers, but any substitution cipher could be included.

    Note that it takes about an hour for the code to run one attack, and it will require multiple trials to crack the cipher.
